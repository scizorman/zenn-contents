# Zenn Contents

The repository that manages my [Zenn](https://zenn.dev) contents.

## How to use

### Create a new article

```console
$ npm run new:article
```

### Create a new book

```console
$ npm run new:book
```

### Preview

```console
$ npm run preview
```
